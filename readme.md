# TrueAccord Take Home

## Prerequisites

You will need `go` [installed](https://golang.org/doc/install).

## Build

```bash
git clone git@github.com:njnygaard/TrueAccord.git
cd TrueAccord
go build
```

or

```bash
tar -xvpf njnygaard.tar.gz
cd TrueAccord
go build
```

## Run

```bash
./TrueAccord debts
```

## Help

```bash
./TrueAccord help
./TrueAccord help debts
```
